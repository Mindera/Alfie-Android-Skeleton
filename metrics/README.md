# Weekly AI Metrics Reports

This directory contains automatically generated weekly reports tracking AI development productivity metrics.

Reports are generated by the GitHub Actions workflow and submitted as Pull Requests.

## Report Format

Each report includes:
- Time to merge metrics
- CI/CD performance data
- Human interaction statistics
- AI vs human productivity comparison
- Code change metrics
- Insights and recommendations

## Automation

Reports are automatically generated every Monday at 9:00 AM UTC via the "Weekly AI Metrics Report" GitHub Action.

See `.github/scripts/README.md` for more details.
